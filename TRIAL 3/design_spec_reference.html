Required Features — Both Pages

  • Shared navbar with Day 1–4 links. Active state highlights the current page. DAY 4 has a red "LIVE" badge.
  • Live countdown timer on the Home page — counts down to June 1, 2026 (deadline). Updates every second. Shows Days, Hrs, Min, Sec.
  • YouTube video embed on Day 1 page using the YouTube IFrame Player API. Use URL: https://www.youtube.com/watch?v=dQw4w9WgXcQ
  • Hidden "unlock" box below the video that fades in after exactly 10 seconds of video playback (not page load — video must be playing). CSS fade-in animation required.
  • 4-day card grid on Home page. DAY 4 card has dark background + gold border + "FUNDING OPEN" badge.
  • Prev/Next navigation at the bottom of Day 1 page linking between pages.
  • Fully responsive — single-column layout on mobile for all sections.
  • Deliver as exactly 2 separate HTML files: index.html (Home) and day1.html (Day 1 article).

Design Tokens

  • Primary gold: #C9A84C
  • Background dark: #000 / #0a0a0a
  • Hero background: radial-gradient dark navy to black
  • Grid texture: SVG diagonal lines at ~4% opacity
  • Light section (briefing): #f5f5f0 background, #fff cards
  • Alert bar: #C9A84C background, black text, top of page
  • LIVE badge: #e53e3e (red)
  • Body text (article): #444 on white background
